ATTA Modular Control System

During my time as a field engineer I felt the design of low production test systems was lacking. It would take days to connect a system wire by wire wasting my time and the time of the end user. And it was always a challenge to replace obsolete or broken components since every system was from it's own generation. So I spent my first year at ATTA designing a system that can be installed by someone with no previous system installation experience. Also since every component is standardized we have a custom test system with proper lifecycle management! The first system was installed at a customer's facility early 2019 and has been running production full time since then.
